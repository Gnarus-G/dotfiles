## Dotfile Location

The `dev` script links this dotfiles project to `~/` so always put new dotfiles in this repo.

Paths it links:

- `.agents` → `~/.agents`
- `.config/opencode` → `~/.config/opencode`
- `.config/nvim` → `~/.config/nvim`
- `.config/mcphub` → `~/.config/mcphub`
- And others (see `dev` script for full list)
